Introduction

Visual Studio Code (VS Code) is a lightweight yet powerful source code editor used across many languages.

It includes built-in support for debugging, syntax highlighting, intelligent code completion, and more.

VS Code is available on Windows, macOS, and Linux.

Many developers choose VS Code because it is fast and highly customizable.

The editor can be extended easily with community-built plugins.

  • Free and open source
  • Cross-platform
console.log("Hello VS Code!");
Installation

You can install VS Code from the official website.

Choose the installer that matches your operating system.

After installation, launch VS Code to start customizing your setup.

  • Windows Installer
  • macOS ZIP or Installer
  • Linux package
sudo apt install code brew install --cask visual-studio-code
User Interface

VS Code’s interface includes the Activity Bar, Side Bar, Editor Groups, and Status Bar.

The UI is fully customizable.

You can open multiple editors side by side.

Extensions

Extensions enhance VS Code by adding languages, themes, debuggers, and tools.

The Extensions Marketplace contains thousands of community-authored packages.

You can install, update, or disable extensions anytime.

  • Python extension
  • Prettier formatter
ext install ms-python.python ext install esbenp.prettier-vscode
Keyboard Shortcuts

VS Code provides many shortcuts to improve workflow speed.

Shortcuts can be customized in the settings menu.

Keybindings are stored in a JSON file.

  • Ctrl + Shift + P — Command Palette
  • Ctrl + B — Toggle Sidebar
{ "key": "ctrl+b", "command": "workbench.action.toggleSidebarVisibility" }
